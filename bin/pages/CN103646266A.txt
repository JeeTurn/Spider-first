    <div class="abstract">本发明公开了一种基于二维码的票据生成系统及其识别方法，该系统包括：二维码结构及编码规则建立模组，用于建立二维码符号结构，根据预定的编码规则确定该二维码符号结构中各字符的编码结构，并确定二维码中各个代码及其所代表的含义；接收模组，接收用户输入的票据信息；以及票据生成模组，针对用户输入的票据信息，根据该二维码符号结构并根据已确定的编码结构自动生成二维码于该票据表面，本发明可使得票据/发票在开具、报销、财务核算和审计时大幅提高效率，减少了人力、财力、物力的消耗，克服了市场上机打或者手写票据/发票金额、时间、费用类别的票据/发票在开具和核算上效率低、防伪性不好，财务运营成本高等不足的缺点。</div>

    <p>基于二维码的票据生成系统及其识别方法</p>
    <p>技术领域</p>
    <p>[0001]	本发明涉及一种票据生成系统及其识别方法，特别是涉及一种基于二维码的票据生成系统及其识别方法。</p>
    <p>背景技术</p>
    <p>[0002]	二维条形码，最早发明于日本，它是用某种特定的几何图形按一定规律在平面(二维方向上)分布的黑白相间的图形记录数据符号信息的，在代码编制上巧妙地利用构成计算机内部逻辑基础的比特流的概念，使用若干个与二进制相对应的几何形体来表示文字数值信息，通过图象输入设备或光电扫描设备自动识读以实现信息自动处理。它具有条码技术的一些共性:每种码制有其特定的字符集；每个字符占有一定的宽度；具有一定的校验功能等。同时还具有对不同行的信息自动识别功能、及处理图形旋转变化等特点。</p>
    <p>[0003]	二维码精准、高效、低成本以及防伪的独特优势引发国内、国际百强企业注册潮，与电子商务的紧密结合更是吸引了不少行业门户抢注。二维码广阔的应用空间为产品提供了覆盖范围很广的营销渠道。行业应用的最核心业务就是防伪，主要应用在烟酒、食品、药品、证照等领域；而企业应用最核心的内容就是营销，企业将二维码实施于各种媒介，引导用户快捷、方便地访问企业手机推广页，加强企业与用户的互动，同时二维码特有的主动性决定了统计用户访问效果的精确性，帮助商家进行数据统计和结算，辅助进行营销效果评估，互联网二维码注册中心。</p>
    <p>[0004]	将二维码应用于发票开具和报销核算的比较鲜见。目前大部分使用的发票都是手写或者机打发票金额的纸质发票，其发票开具费用类型单一，发票的功能还不很完善。由于现在的发票大部分是手工开具或机打，在报销、财务核算和审计时需人工分类核算和验证等，造成大量的手工劳动，浪费人力、财力和物力，效率低，并且手工计算容易造成差错，也不利于相关部门进行财务数据分析。另外，现在的发票大小规格不一，发票整理也很繁琐，不易存放保管。</p>
    <p>发明内容</p>
    <p>[0005]	为克服上述现有技术存在的不足，本发明之目的在于提供一种基于二维码的票据生成系统及方法，使得票据/发票在开具、报销、财务核算和审计时大幅提高效率，减少了人力、财力、物力的消耗，克服了市场上机打或者手写票据/发票金额、时间、费用类别的票据/发票在开具和核算上效率低、防伪性不好，财务运营成本高等不足的缺点。</p>
    <p>[0006]	为达上述及其它目的，本发明提出一种基于二维码的票据生成系统，至少包括:</p>
    <p>[0007]	二维码结构及编码规则建立模组，用于建立二维码符号结构，根据预定的编码规则确定该二维码符号结构中各字符的编码结构，并确定二维码中各个代码及其所代表的含义；</p>
    <p>[0008]	接收模组，接收用户输入的票据信息；以及</p>
    <p>[0009]	票据生成模组，针对用户输入的票据信息，根据该二维码符号结构并根据已确定的编码结构自动生成二维码于该票据表面。</p>
    <p>[0010]	进一步地，该票据生成系统还包括一防伪模组，用于形成防伪二维码，并将该防伪二维码印制于该票据上。</p>
    <p>[0011]	进一步地，该二维码符号结构包括空白区、位置探测图形、位置探测分隔符、校正图形、定位图形、格式信息、版本信息以及数据区域和纠错码字，该空白区为二维码外围的白色区域，该位置探测图形和该位置探测分隔符为用于对二维码定位，该校正图形为用于二维码形状矫正，该定位图形为在二维码上定义了网格，该格式信息表示二维码的的纠错级别，该数据区域为使用黑白的二进制网格编码内容，该版本信息为二维码的规格，该纠错码字为用于修正二维码损坏带来的错误。</p>
    <p>[0012]	进一步地，该二维码符号结构包括三个位置探测图形，分别位于图像的左上角、右上角、左下角，每个位置探测图形由3个重叠的同心的正方形组成，每个位置探测图形的模块宽度比为1:1:3:1:1。</p>
    <p>[0013]	进一步地，该校正图形为3个重叠的同心正方形，由5 X 5个的深色模块、3 X 3个的浅色模块以及位于中心的一个深色模块组成。</p>
    <p>[0014]	进一步地，该二维码符号结构可以横向和纵向两个方位同时表达信息，由某种特定的图形按规律在平面上记录信息。</p>
    <p>[0015]	进一步地，该编码规则为QR Code 二维码编码规则或者Codel6K、Code49、PDF417、Aztec、Maxi Code&gt; Data Matrix、Postnet、BP04_State 编码规则。</p>
    <p>[0016]	进一步地，QR Code 二维码为在一个矩形空间通过黑、白像素在矩阵中的不同分布进行编码，在矩阵相应元素位置上，用点的出现表示二进制“1”，点的不出现表示二进制的“0”，点的排列组合确定了矩阵式二维条码所代表的意义。</p>
    <p>[0017]	进一步地，该二维码开头的第I个符号为模式指示符，第2个数字字符为发票类型代码，第3个数字字符为税务局代码，第4-9个数字字符为票据开具企业财税所属行政区域代码，第10-11位数字字符为票据开具单位所属行业代码，第12-17位数字字符为票据的开具日期，第18-19位为票据的费用类型代码，第20位数字字符为该票据可开具的最大金额，第21-27位数字字符为该票据的金额。</p>
    <p>[0018]	为达到上述目的，本发明还提供一种基于二维码的票据的识别方法，包括如下步骤:</p>
    <p>[0019]	步骤一，基于用户输入的信息及二维码生成规则生成二维码于票据表面；</p>
    <p>[0020]	步骤二，二维码扫描器扫描该票据的二维码获取二维码信息；</p>
    <p>[0021]	步骤三，译码器根据预定的二维码编码规则自动翻译出二维码中的各代码信息，从而获得票据信息。</p>
    <p>[0022]	进一步地，于步骤一中，将发票防伪码同时隐含于该票据的二维码中；于步骤二，该二维码扫描器还扫描该票据的防伪数据块；于步骤三，该译码器还对该防伪数据块根据预设规则进行翻译，并将翻译的结果查询数据库。</p>
    <p>[0023]	进一步地，该译码器将翻译结果传输给计算机，利用计算机通过网络查询网络数据库，以确定该票据的真伪。</p>
    <p>[0024]	进一步地，于步骤一中，将防伪码和票据信息各生成一个二维码，分别用于识别票据信息与防伪信息，于步骤二，该二维码扫描器还扫描该票据的防伪数据块；于步骤三，该译码器还对该防伪数据块根据预设规则进行翻译，并将翻译的结果查询数据库。</p>
    <p>[0025]	与现有技术相比，本发明一种基于遗传算法的神经网络优化系统及方法通过利用遗传算法对神经网络的权值和阈值进行快速优化，在解空间中定位出一个较好的搜索空间，并将此空间作为神经网络搜索的初始权值和阈值，然后利用神经网络的局部搜索能力在这个小的解空间中搜索出最优解，能极大地提高搜索时间，得到较好的权值和阈值分布。</p>

    <p>[0053]	QR Code 二维码是在一个矩形空间通过黑、白像素在矩阵中的不同分布进行编码。在矩阵相应元素位置上，用点(方点、圆点或其他形状)的出现表示二进制“1”，点的不出现表示二进制的“0”，点的排列组合确定了矩阵式二维条码所代表的意义</p>
    <p>[0054]	QR Code 二维码包括:空白区、位置探测图形、位置探测分隔符、校正图形、定位图形、格式信息、版本信息以及数据区域和纠错码字，输入信息时采用字母数字模式，将输入的数据分为两个字符一组，用11位二进制表示。将前面字符的值乘以45与第二个字符的值相加，将所得的结果转换为11位二进制数。如果输入的数据的字符数不是2的整数倍，将最后一个字符编码为6位二进制数。将所得的二进制数据连接起来并在前面加上模式指示符和字符计数指示符，按表I的规定在字母数字模式中，字符计数指示符的长度为9、11或13位，其定义见表3。将输入的字符数编码为9、11或13位二进制数，放在模式指示符之后，二进制数据序列之前。图3为本发明较佳实施例中针对酒店餐饮业中酒店住宿费举例设计的发票中的一种QR Code 二维码的示意图</p>

    <p>[0082]	需说明的是，行业代码前包括行业代码与二维码、二维码防伪模块与二维码由票据发行财政部门制作时印制(或者由相关管理机构授权开具发票单位，通过联网系统软件打印)，其它二维码由票据开具单位通过软件打印。</p>

    <p>[0084]	较佳的，为使票据具有防伪功能，本发明之基于二维码的票据生成系统还包含一防伪模组13，在输入二维码信息之后，防伪模组13会建立一个新的数据模块，单独用于二维码的防伪。</p>
    <p>[0085]	本发明中的防伪模组直接将防伪码制作成二维码与编码的形式，在人们辨别票据/发票的真伪时，直接对票据/发票上面的二维码进行扫描或者读取，可简单、轻松、快捷地辨别票据/发票的真伪。本发明中的防伪模组由12位0-9的数字字符组成，并且条码中的12位代码是由0-9中10个数字随机产生，只能使用一次，伪劣票据/发票制造商无法制作出，从而达到防伪的目的。图8是本发明中一种随机产生的防伪二维码的示意图，由12位0-9的随机数组成。</p>
    <p>[0086]	图9为传统发票票样示意图，图10为本发明之二维码票据的票据示意图，发票右上端为二维码，二维码里面包含输入的信息以及防伪模块，本发明通过扫描发票中的二维码，可以将二维码中的12位防伪码翻译，通过译码器的翻译之后，传输给电脑软硬件系统，再通过网络来辨别该发票的真伪，操作简单、输入精确、效率高减少了人力资源的消耗。</p>
    <p>[0087]	图11为本发明一种基于二维码的票据的识别方法的步骤流程图。如图11所示，本发明一种基于二维码的票据的识别方法，包括如下步骤:</p>
    <p>[0088]	步骤101，基于用户输入的信息及二维码生成规则生成二维码于票据表面；</p>
    <p>[0089]	步骤102，二维码扫描器扫描该票据的二维码获取二维码信息；</p>
    <p>[0090]	步骤103，译码器根据预定的二维码编码规则自动翻译出二维码中的各代码信息，从而获得票据信息，如票据的发行地区、财税类型、票据开具单位所属行业、费用类别、金额等信息。</p>
    <p>[0091]	较佳的，于步骤101中，将发票防伪码同时隐含于票据的二维码中，同时于步骤102中，该二维码扫描器还扫描该票据的防伪数据块，步骤三中的译码器还对该防伪数据块根据预设规则进行翻译，并将翻译的结果查询数据库。</p>
    <p>[0092]	于步骤103后，该译码器将翻译结果传输给计算机，利用计算机通过网络查询网络数据库，以确定该票据的真伪。</p>
    <p>[0093]	于步骤101中，生成二维码票据的方案如下:方案1，依据某种规则，将用户输入的所有信息转化成数字代码，然后将数字代码转化成二维码，生成于票据表面；方案2，依据某种规则，将用户输入的所有信息直接转化成二维码，然后生成于票据表面；方案3，依据某种规则，将防伪码和票据信息一同直接生成一个二维码，用于识别票据信息与防伪信息；方案4，将防伪码和票据信息各生成一个二维码，分别用于识别票据信息与防伪信息。</p>
    <p>[0094]	综上所述，本发明一种基于二维码的票据生成系统及其识别方法，通过生成预定编码规则的二维码，该二维码可以被二维码扫描器扫描，被译码器翻译，被计算机硬件、软件所识别，进一步显示出该票据/发票上面的费用类型、金额、开具时间，由于二维码采集信息量大、灵活实用、易于制作和输入速度快的特点，使得票据/发票在开具、报销、财务核算和审计时大幅提高效率，减少了人力、财力、物力的消耗，克服了市场上机打或者手写票据/发票金额、时间、费用类别的票据/发票在开具和核算上效率低、防伪性不好，财务运营成本闻等不足的缺点。</p>
