    <p>伊维菌素和茶皂素的增效农药组合物</p>
    <p>技术领域</p>
    <p>[0001]	本发明属于农药技术领域，其特征是涉及一种增效的农药组合物，特别是该组合物的活性成分一方面是伊维菌素，另一方面是茶皂素。</p>
    <p>背景技术</p>
    <p>[0002]	伊维菌素（Ivermectin)是以阿维菌素为先导化合物，通过双键氢化，结构优化而开发成功的新型合成农药。作为农用抗菌素的结构优化产物，与母体阿维菌素相比，不但保留了其驱虫和杀满活性，而且安全性更高，不易产生抗性，为蔬菜、水果、棉花等的生产提供了一个高效、高安全性及与环境相容性好的农药杀虫杀螨剂。伊维菌素具有胃毒和触杀作用，并有横向渗透传导作用，对昆虫、螨类和线虫都有驱杀作用。</p>
    <p>[0003]	在农药制剂中恰当地使用增效剂，最大限度地降低伊维菌素有效成分的使用量以充分保护该高效、高安全性及与环境相容性好的农药品种，延长其使用寿命，而同时不影响制剂的药效，是伊维菌素农药可持续发展中亟待解决的问题。</p>
    <p>[0004]	茶皂素（Tea saponin, TS)又名茶皂苷、茶皂角苷、茶皂草苷，是一种性能良好的天然表面活性剂，不仅具有良好的乳化、发泡、分散、渗透、润湿等活性作用，而且还具有消炎、 止痛、杀菌、止痒等药理作用。茶皂素是水剂或可溶性粉剂农药的优良助剂，起到对农药的增效作用。茶皂素能自动降解，它在分解过程中，不会对农药的化学性能产生影响。茶皂素可以作为农药活性成分进行使用，但茶皂素速效性差、施用剂量大，单独使用效果不好。</p>
    <p>[0005]	在本发明中，茶皂素将作为农药活性成分之一被用于研究其与另一有效成分伊维菌素的共毒增效作用，为发掘茶皂素的新用途和高效安全地利用伊维菌素农药提供新出路。以伊维菌素与茶皂素为活性成分的增效农药组合物未见专利报道。</p>
    <p>发明内容</p>
    <p>[0006]	本发明的目的是提供一种农药技术领域的伊维菌素和茶皂素的增效农药组合物， 以期提供一种高效、低毒、广谱及不易引致抗药性的的环境友好型农药组合物。</p>
    <p>[0007]	本发明提供的农药组合物，其特征在于：有效成分伊维菌素和茶皂素的质量比为 1 ： 0.01&#12316;1 ： 100，优选为1 ： 0. 1&#12316;1 ： 50，最佳为1 ： 1&#12316;1 ： 10，其余为农药助剂， 按照本领域技术人员已知的方法加工配制成在实际应用中适合农业上使用的任意制剂。所述的助剂根据制剂具体需要而确定。</p>
    <p>[0008]	所述的茶皂素可以是提取物、茶皂素纯品或者其混合物及其制剂。</p>
    <p>[0009]	本发明的积极效果在于：拓宽害虫防治谱，一次用药可兼治疗两种或多种同时发生的害虫；互补了作用机制，在茶皂素杀虫的同时增强药液的展着、润湿作用，对伊维菌素具有显著的增效作用，减少其使用量，延长使用寿命；天然源农药茶皂素的使用，在组合物中同时可以充当表面活性剂的角色，大大减少其它表面活性剂的添加，更加环保。</p>
    <p>具体实施方式[0010]	为了更好地理解本发明的实质，下面略举实例来进一步说明本发明组合物的制备和共毒效应应用，但本发明的内容并不局限于此。</p>
    <p>[0011]	实施例1 ：配制20%伊维菌素&#183;茶皂素微乳剂</p>
    <p>[0012]	杀虫有效成分重量百分比（％ )为：伊维菌素5. 0，茶皂素15. 0 ；</p>
    <p>[0013]	助剂各成分重量百分比（％)为：乳化剂：聚氧乙烯烷基胺为10.0;溶齐IJ: SolvessolOO为15. 0 ；助溶剂=N-甲基吡咯烷酮为5. 0 ；防冻剂：丙三醇为2. 0 ；余量去离子水。</p>
    <p>[0014]	按配比将伊维菌素溶于溶剂中，将茶皂素溶于水中，将乳化剂加入到伊维菌素溶液中，充分搅拌均&#21243;，加入其余助剂、茶皂素水溶液，投入到&#21243;化器中予以强烈的混合并&#21243;化即得。</p>
    <p>[0015]	实施例2 ：配制10. 5%伊维菌素&#183;茶皂素水乳剂</p>
    <p>[0016]	杀虫有效成分重量百分比（％ )为：伊维菌素3. 5，茶皂素7. 0 ；</p>
    <p>[0017]	助剂各成分重量百分比（％ )为：乳化剂：TX-10为4. 5 ；增稠剂：阿拉伯树胶为 2. 0，黄原胶为1. 0 ；溶剂：柴油为11. 2. 0 ；防冻剂：山梨酸为5. 0 ；消泡剂：椰子油EO-PO聚合物为0. 2 ；防腐剂：山梨酸钾为2. 0 ；余量去离子水。</p>
    <p>[0018]	按配比将伊维菌素和茶皂素原药溶于溶剂中，将乳化剂、助溶剂加入到伊维菌素和茶皂素溶液中，充分搅拌均&#21243;，将上述混合物缓慢加入水中，充分搅拌即得。</p>
    <p>[0019]	实施例3 ：配制11%伊维菌素&#183;茶皂素悬浮剂</p>
    <p>[0020]	杀虫有效成分重量百分比（％ )为：伊维菌素1. 0，茶皂素10. 0 ；</p>
    <p>[0021]	助剂各成分重量百分比（％ )为：润湿分散剂：WD-C为11. 0 ；增稠剂：聚乙烯醇吡咯烷酮为1. 2，黄原酸胶为1. 0 ；消泡剂：有机硅酮为1. 5 ；防冻剂：丙三醇为5. 6 ；防腐剂： 柠檬酸钠为0. 5 ；余量去离子水。</p>
    <p>[0022]	按配比将伊维菌素和茶皂素原药进行粉碎，加入除增稠剂以外的各种助剂和水， 一起放入高剪切乳化机中混合均&#21243;，加入增稠剂进一步混合均&#21243;，经磨砂机研磨，镜检合格后即得。</p>
    <p>[0023]	实施例4 ：配制12. 5%伊维菌素&#183;茶皂素可湿性粉剂</p>
    <p>[0024]	杀虫有效成分重量百分比（％ )为：伊维菌素为2. 5，茶皂素为10. 0 ；</p>
    <p>[0025]	助剂各成分重量百分比（％ )为：分散剂：烷基磺酸盐20. 0 ；余量载体：高岭土。</p>
    <p>[0026]	按配比将伊维菌素和茶皂素原药进行气流粉碎后加入到混合器中，按配比分别向混合器中加入分散剂、填料，充分搅拌均&#21243;即得。</p>
    <p>[0027]	实施例5 ：配制80. 8%伊维菌素&#183;茶皂素水分散性粒剂</p>
    <p>[0028]	杀虫有效成分重量百分比（％ )为：伊维菌素80. 0，茶皂素0. 8 ；</p>
    <p>[0029]	助剂各成分重量百分比（％ )为：润湿剂：Lx-C为3.6 ；崩解剂：无水硫酸钠为 2. 5 ；分散剂：ΝΝ0为3. 0，余量载体：活性白土。</p>
    <p>[0030]	按配方见伊维菌素和茶皂素原药与其他辅助材料一起用气流粉碎等进行干式微细粉碎，得到粉状的可湿性粉剂后，再将该水分散型粒剂造粒即得。</p>
    <p>[0031]	实施例6 ：配制10. 伊维菌素&#183;茶皂素水剂</p>
    <p>[0032]	杀虫有效成分重量百分比（％ )为：伊维菌素0. 1，茶皂素10. 0 ；</p>
    <p>[0033]	助剂各成分重量百分比（％ )为：表面活性剂：十二烷基苯磺酸钠为5. 0，苯甲酸钠为1.0，余量去离子水。</p>
    <p>[0034]	实施例7 ：配制11 %伊维菌素&#183;茶皂素可溶性液剂</p>
    <p>[0035]	杀虫有效成分重量百分比（％ )为：伊维菌素0. 65，茶皂素10. 35 ；</p>
    <p>[0036]	助剂各成分重量百分比（％ )为：表面活性剂：失水山梨醇聚氧乙烯醚13. 0 ；防冻剂：丙三醇2. 5 ；增溶剂：乙酸乙酯15. 0 ；溶剂补至100。</p>
    <p>[0037]	下面用室内联合毒力测定例子对本发明组合物的增效作用进行说明，但在实际应用中本发明不局限于以下具体试验例。</p>
    <p>[0038]	本发明对伊维菌素和茶皂素的增效组合物进行的室内毒力试验采用了常规方法， 而数据计算和处理按照如下方法进行：</p>
    <p>[0039]	用Abbott公式校正各处理的死亡率，用DPS软件计算LC5tl和毒力回归方程。参照 NY/T1154. 7-2006采用孙云沛的方法求出各混配比例的共毒系数（CTC)，并评价二者混用的增效作用。</p>
    <p>[0040]	判断标准：CTC彡80，表现为拮抗作用；CTC ：80&#12316;120，表现为相加作用； CTC彡120，表现为增效作用。</p>
