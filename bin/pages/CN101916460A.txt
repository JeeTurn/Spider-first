    <div class="abstract">本发明公开了基于移动通讯网络的车位管理系统及停车收费方法，属于车库管理技术领域，本发明的车位管理系统，与接入系统和门禁管理系统相连，包括通信模块、车位调度模块、支付模块、门禁控制模块，移动终端向车位管理系统发送停车请求；所述车位管理系统为移动终端分配车位及进行验证，验证通过后控制所述门禁管理系统开启车库入口的门禁，启动对该移动终端的停车计费；移动终端在离开时向车位管理系统发送离开请求；车位管理系统生成所述移动终端号码的计费信息发送到计费系统进行扣费，向移动终端发送扣费信息，通知门禁管理系统开启车库出口的门禁。本发明通过移动终端可自助完成停车收费管理，具有操作方便，安全高效的特点。</div>

    <p>基于移动通讯网络的车位管理系统及停车收费方法</p>
    <p>技术领域</p>
    <p>[0001]	本发明涉及车库管理技术领域，尤其涉及基于移动通讯网络的车位管理系统及停 车收费方法。</p>
    <p>背景技术</p>
    <p>[0002]	随着汽车拥有量的大幅提升，人们对停车的需求使得各种场所的停车库的使用频 率大为增加。对于收费车库，尤其是商业或办公场所等收费停车场，尽管大多都配置了车位 管理系统，可以实现自动计算停车时间，统计空余车位等功能。但目前通常采用人工收费方 式，收费效率低，这种收费往往多为一次性的小额消费，在车辆出入频繁的停车场往往会造 成堵塞等严重后果。</p>
    <p>[0003]	随着移动通信网络的迅速发展，手机已成为人们生活中的随身携带的必需品，移 动支付技术也得到广泛的应用。因而，这就产生了使用手机实现控制车库门禁和收费的需 求，如何以实时交互的方式实现通过手机自助出入门禁系统以及自助付费成为需要解决的 技术问题。</p>
    <p>发明内容</p>
    <p>[0004]	本发明所要解决的技术问题在于，提供一种基于移动通讯网络的车位管理系统及 停车收费方法，用以通过移动终端实现自助停车收费。</p>
    <p>[0005]	为了解决上述问题，本发明提出了一种车位管理系统，其与接入系统和门禁管理 系统相连，包括：通信模块、车位调度模块、支付模块、门禁控制模块，其中：</p>
    <p>[0006]	所述通信模块，通过接入系统与移动终端进行通信；</p>
    <p>[0007]	所述车位调度模块，用于在通信模块收到停车请求或验证请求时为该移动终端分 配车位，通过通信模块将所述车位信息反馈给移动终端；</p>
    <p>[0008]	所述门禁控制模块，用于在通信模块收到停车请求时生成验证信息反馈给移动终 端；用于在通信模块收到验证请求时对移动终端上报的验证信息进行验证，在验证通过后， 用于控制门禁管理系统开启车库入口的门禁；在通信模块收到离开请求时，用于控制门禁 管理系统开启车库出口的门禁；</p>
    <p>[0009]	所述支付模块，用于在门禁控制模块对移动终端的验证信息验证通过后，记录移 动终端的号码及当前时间，启动对该移动终端的停车计费；用于在通信模块收到离开请求 时，生成所述移动终端号码的计费信息，根据计费信息进行扣费，由通信模块经接入系统向 移动终端发送扣费信息。</p>
    <p>[0010]	所述车位调度模块，还用于在所述通信模块收到移动终端的停车查询请求后，获 取移动终端的当前位置并查询移动终端当前位置附近的停车场信息及对应的车位信息，并 将具有空闲车位的停车场信息反馈给移动终端。</p>
    <p>[0011]	所述通信模块是采用非结构化补充数据业务USSD或短消息业务SMS的方式通过 接入系统与移动终端进行通信联络。</p>
    <p>4[0012]	所述车位调度模块，还用于在为移动终端分配车位后将该停车场对应的空闲车位 数减1 ；在移动终端离开后，将该停车场对应的空闲车位数加1。</p>
    <p>[0013]	所述门禁控制模块在通信模块收到停车请求时生成的验证信息包括验证码及该 车位的预留时间提示信息。</p>
    <p>[0014]	本发明还提供一种基于移动通讯网络的车位管理系统的停车收费方法，包括：</p>
    <p>[0015]	移动终端通过接入系统向与接入系统和门禁管理系统相连的车位管理系统发送 停车请求；</p>
    <p>[0016]	所述车位管理系统为移动终端分配车位及进行验证，验证通过后控制所述门禁管 理系统开启车库入口的门禁，启动对该移动终端进行停车计费；</p>
    <p>[0017]	所述移动终端在离开时通过接入系统向车位管理系统发送离开请求；</p>
    <p>[0018]	所述车位管理系统生成所述移动终端号码的计费信息发送到计费系统进行扣费， 通过接入系统向移动终端发送扣费信息，通知门禁管理系统开启车库出口的门禁。</p>
    <p>[0019]	所述车位管理系统与移动终端采用非结构化补充数据业务USSD或短消息业务 SMS的方式进行通信联络。</p>
    <p>[0020]	所述车位管理系统在为移动终端分配车位后，进一步将该停车场对应的空闲车位 数减1 ；在移动终端离开后，将该停车场对应的空闲车位数加1。</p>
    <p>[0021]	所述移动终端向车位管理系统发送停车请求时，是根据自身存储的停车场信息、 或直接输入的停车场信息，或者向所述车位管理系统查询后获取的当前位置附近的具有空 闲车位的停车场信息来发送停车请求。</p>
    <p>[0022]	所述车位管理系统是在收到移动终端的停车查询请求后，获取移动终端的当前位 置，查询获取移动终端当前位置附近的停车场信息及对应的车位信息，并将具有空闲车位 的停车场信息反馈给移动终端。</p>
    <p>[0023]	本发明的车位管理系统及停车收费方法中，移动终端采用USSD和SMS接入方式通 过接入系统接入车位管理系统，由车位管理系统控制车辆停车及离开时的门禁开启，以及 在离开时生成计费信息，送到计费系统进行扣费，完成自助式的停车收费管理，具有操作方 便，安全高效的特点。本发明由于采用手机交互方式自助实现停车收费，可以省掉额外的人 工收费操作；本发明不需要在汽车上加装额外的传感器，利用驾驶员随身携带的移动终端 便可实现门禁的自动开启；本发明由于采用移动支付，使得停车费的收取更加便捷和安全。</p>

    <p>[0027]	为使本发明的目的、技术方案和优点更加清楚，以下结合附图对本发明作进一步 地详细说明。</p>
    <p>[0028]	本发明的基于移动通讯网络的车位管理系统及停车收费方法，利用移动通信网 络，非结构化补充数据业务 USSD (Unstructured Supplemenary Service Data) USSD 和 / 或短消息业务SMS(Short Messaging Service)技术，以及移动支付技术，基于移动通讯网络 实现车库管理，在移动通信网（即接入系统）中通过交互实现停车和收费，提高车库使用效 率和便捷性，降低车库管理的人力成本。</p>
    <p>[0029]	所述非结构化补充数据业务 USSD (Unstructured Supplementary Service Data) 是一种基于全球移动通信网络的、实时在线的新型交互会话数据业务，面向连接，具有可靠 性高的特点。USSD目前技术成熟，它基于用户识别模块SIM (Subscriber Identity Module) 卡，利用网络的信令通道传送数据。当用户访问USSD业务时，该对话通路保持连接，直到对 话结束。此外，由于提供了通过HLR(归属位置寄存器，Home Location Register)直达外 部业务节点的传输USSD字串透明通路，用户还可访问位于GSM网络以外的业务节点。</p>
    <p>[0030]	所述USSD技术具有如下特点：（1)快速，在交互式应用中由于只需建立一次连接， 减少了多次建立信令连接的时延，来回一次的响应时间比短消息业务短，最短是短消息的 七分之一；（2)方便，用户无需进行逐级菜单选择，而是直接输入USSD命令字串；(3)支持 漫游，USSD的每一条命令都要经过HLR，因此用户漫游时以基于USSD业务的同样方式运行； (4)广泛适用，USSD可在目前所有的手机上运行；（5)功能强，能以对话方式和信息业务菜 单方式提供服务。</p>
    <p>[0031]在第三代合作伙伴计划	3GPP(Third Generation Partnership Project)中也 有对USSD的描述，WCDMA和TD-SCDMA都是遵循的3GPP标准，也就是说USSD技术在全球 移动通讯系统（Global System for Mobile Communications，GSM)、宽带分码多工存取 (Wideband Code Division Multiple Access, WCDMA)和时分同步码分多址接入（Time Division-Synchronous Code Division Multiple Access, TD-SCDMA)网络中被广泛的支 持。</p>
    <p>[0032]	所述短消息业务SMS(Short Messaging Service)是现在普及率最高的一种业 务。目前，这种短消息的长度被限定在140字节之内，这些字节可以是文本的。SMS以简单 方便的使用功能受到大众的欢迎。所述SMS业务具有实现简单、发送通道畅通等优点。在 3GPP (Third Generation Partnership Project)中也有对 SMS 的描述，WCDMA 和 TD-SCDMA 都是遵循的3GPP标准，也就是说SMS技术在全球移动通讯系统（Global System for Mobile Communications, GSM)、宽带分石马多工存取(Wideband Code Division Multiple Access, WCDMA)和时分同步码分多址接入（Time Division-Synchronous Code Division Multiple Access, TD-SCDMA)网络中被广泛的支持。</p>
    <p>[0033]	所述非结构化补充数据业务（USSD)与短消息业务（SMS)都是通过网络的信令通 道传递的，不同之处在于短消息属于存储转发方式，USSD则属于面向对话方式。</p>

    <p>[0048]	如图2和3所示，本发明的基于移动通讯网络的车位管理系统中停车收费流程如 下：</p>
    <p>[0049]	步骤一：驾驶员利用手机通过USSD或者SMS方式经接入系统向车位管理系统发起 查询周边车库的信息；</p>
    <p>[0050]	步骤二 ：车位管理系统根据基站信息或手机上报的位置信息，定位手机大致所在 位置，将驾驶员所在位置周边的停车场地址和空闲车位信息发送到手机上；</p>
    <p>[0051]	步骤三：驾驶员根据车位管理系统反馈的停车场信息，选择到其中一个停车场停 车，再通过USSD或者SMS业务向系统发起到某一停车场停车请求，与车位管理系统建立需 要进入车库停车会话；</p>
    <p>7[0052]	步骤四：车位管理系统根据驾驶员所选停车场，检索空闲车位信息，将该停车场空 位数减1，并将该停车场门禁控制模块生成的验证码发送给驾驶员的手机，并可提示该验证 码15分钟内有效；</p>
    <p>[0053]	步骤五：驾驶员驾车到车库门禁处，通过手机终端发送验证码到车位管理系统，验 证码验证通过后，车位管理系统通知门禁管理系统开启车库门禁，并记录该手机的号码和 当前时间，开始计时停车；</p>
    <p>[0054]	步骤六：如图3所示，驾驶员需要开车离开车库时，手机发送要离开车库的请求消 息，发送时手机终端先发送需要离开的请求到接入系统，接入系统再将该手机号码和驾车 离开的请求发送给车位管理系统。</p>
    <p>[0055]	步骤七：车位管理系统收到主叫号码和驾车离开的请求后，获取数据库中该手机 号码的停车信息，根据驶入车库的时间自动生成计费信息，发送到计费系统（所述计费系 统可以是运营商计费系统也可是专用计费系统）进行扣费，然后注销该手机号码的停车信 息，空闲车位数加1，并发送驾驶员的手机的停车和扣费信息到接入系统，同时通知门禁管 理系统开启车库出口的门禁。</p>
    <p>[0056]	步骤八：接入系统收到车位管理系统的停车和扣费信息后，反馈给手机终端。</p>
    <p>[0057]	步骤九：手机终端收到停车和扣费信息通知，结束会话。</p>
    <p>[0058]	本发明的停车管理系统，除可应用于手机外，还可以适用于具有通信功能的车载 GPS定位仪或移动电脑或导航仪或车载电视终端。</p>
    <p>[0059]	移动终端向车位管理系统发送停车请求时，一方面可以向所述车位管理系统查询 后获取的当前位置附近的具有有效空闲停车位的停车场信息来发送停车请求，也可以根据 自身存储的停车场信息或直接输入的停车场信息来向车位管理系统进行停车请求。</p>
    <p>[0060]	以上所述仅为本发明的实施例而已，并不用于限制本发明，对于本领域的技术人 员来说，本发明可以有各种更改和变化。凡在本发明的精神和原则之内，所作的任何修改、 等同替换、改进等，均应包含在本发明的权利要求范围之内。</p>
