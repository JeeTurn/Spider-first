    <p>微乳液油墨清洗剂及其制备方法</p>
    <p>技术领域</p>
    <p>[0001]	本发明属于印刷油墨清洗技术领域， 特别是微乳液型油墨清洗剂及其制备方法。 背景技术</p>
    <p>[0002]	在印刷行业中，印刷机在执行印刷任务过程中通常会沾染油墨，油墨中的干性油 在催干剂作用下会迅速氧化、聚合并干燥形成油膜，影响印刷品的质量，因此要经常对印刷 机进行清洗，以保证印刷机正常工作。传统的印刷油墨清洗剂以汽油和煤油为主要溶剂，它 们具有闪点低、易挥发、且可燃等特点，会影响印刷企业安全生产，也不利于员工的身体健 康。于是，人们把目光转向新型油墨清洗剂的开发。</p>
    <p>[0003]	现有新型油墨清洗剂的研究主要分为三类：混合溶剂型油墨清洗剂、水基型油墨 清洗剂和乳液型油墨清洗剂。混合溶剂型油墨清洗剂在配方中加入了高沸点的有机溶剂， 由于其具有可燃性，对于印刷厂中堆放的纸张等易燃物存在火灾隐患。水基型油墨清洗剂 以水为主要溶剂，利用表面活性剂和碱性助剂的共同作用去除油墨污垢。这类清洗剂虽然 不可燃，但也存在缺陷：清洗对象范围狭窄，有些只能清洗墨辊；适用于浸泡式清洗，对擦 洗、喷淋效果欠佳。乳液型油墨清洗剂为保持对油墨有良好的清洗作用，采用汽油、煤油等 有机溶剂做油相，只减少有机溶剂在溶液中的比例，不能彻底消除对环境的污染。</p>
    <p>发明内容</p>
    <p>[0004]	本发明的目的在于克服上述油墨清洗剂的缺点，提供一种除油墨污垢速度快、清 洗效果好、安全性好的微乳液型油墨清洗剂。</p>
    <p>[0005]	本发明组成包括阴离子表面活性剂、非离子表面活性剂、助表面活性剂、有机酯 类、缓蚀剂、渗透剂和纯水。</p>
    <p>[0006]	在清洗剂中各组分的重量百分比分别为：阴离子表面活性剂占0. 2-5. 0%，非离 子表面活性剂占1. 0-15. 0 %，助表面活性剂占5. 0-30. 0 %，有机酯类占0. 5-20. 0 %，缓蚀 剂占0. 1-5. 0%，渗透剂占0. 2-5. 0%，其余为纯水。</p>
    <p>[0007]	本发明的油墨污垢去除机理为：由于油墨污垢是以连接料形成的油膜粘附在被清 洗表面的，清洗剂中的表面活性剂和助表面活性剂会以极性基团朝向水连续相的形式在油 墨污垢和水的界面上同时吸附，使该界面上的表面张力迅速降低至10_2mN/m，瞬间润湿油墨 污垢表面，渗透剂协助表面活性剂和助表面活性剂渗透到油墨污垢中，润湿被清洗表面，将 油墨污垢中由连接料形成的长链剪断，使油墨污垢“蜷缩”起来，借助擦拭，脱离机器表面。</p>
    <p>[0008]	本发明产品可用于清洗印刷机、墨辊和胶皮布，其去除油墨污垢速度快，清洗效果 好，生产成本低，不挥发，不可燃，不污染环境。</p>
    <p>[0009]	本发明所述阴离子表面活性剂为烷基磺酸盐表面活性剂、全氟烷基磺酸盐表面活 性剂或脂肪醇聚氧乙烯醚硫酸盐表面活性剂中的任意一种。</p>
    <p>[0010]	所述非离子表面活性剂为脂肪醇聚氧乙烯醚非离子表面活性剂、烷基酚聚氧乙烯 醚非离子表面活性剂、烷基醇酰胺非离子表面活性剂中的任意一种。[0011]	所述助表面活性剂为一元醇、多元醇或醇醚中的任意一种。</p>
    <p>[0012]	所述有机酯类为乙酸酯。</p>
    <p>[0013]	所述缓蚀剂为有机胺类。</p>
    <p>[0014]	所述渗透剂为脂肪醇聚氧乙烯醚（加成数为4-8)。</p>
    <p>[0015]	前述微乳液油墨清洗剂的制备方法为：</p>
    <p>[0016]	在室温下，先将阴离子表面活性剂、非离子表面活性剂、有机胺、渗透剂加入纯水 中搅拌均&#21243;配制成溶液，然后在搅拌条件下，加入有机酯进行乳化，搅拌20min士2min，待乳 状液稳定后，继续加入助表面活性剂，再搅拌30min士2min，制得微乳液油墨清洗剂。</p>
    <p>[0017]	投料时，各组分在清洗剂中的重量百分比分别为：阴离子表面活性剂占 0. 2-5. 0 %，非离子表面活性剂占1. 0-15. 0 %，助表面活性剂占5. 0-30. 0 %，有机酯类占 0. 5-20. 0 %，缓蚀剂占 0. 1-5. 0 %，渗透剂占 0. 2-5. 0 %。</p>
    <p>具体实施方式</p>
    <p>[0018]	实施例1 ：</p>
    <p>[0019]	在室温下，先将2g脂肪醇聚氧乙烯硫酸钠、3g脂肪醇聚氧乙烯醚（加成数为9)、 2g壬基酚聚氧乙烯醚、1 g十二烷基苯磺酸钠、2g烷基醇酰胺、2g三乙醇胺和2g渗透剂脂肪 醇聚氧乙烯醚加入73g纯水中搅拌，溶解，配制成溶液，然后在搅拌条件下，加入5g乙酸正 丁酯进行乳化，搅拌20min，待乳状液稳定后，继续加入8g乙二醇乙醚，搅拌30min，制得微 乳液油墨清洗剂100g。</p>
    <p>[0020]	实施例2 ：</p>
    <p>[0021 ] 在室温下，先将1 g全氟丁基磺酸钾、3g脂肪醇聚氧乙烯醚（加成数为15)、2g辛基 酚聚氧乙烯醚、2g十六烷基苯磺酸钠、3g羟乙基乙二胺和3g脂肪醇聚氧乙烯醚加入71g纯 水中搅拌，溶解，配制成溶液，然后在搅拌条件下，加入3g乙酸戊酯进行乳化，搅拌20min， 待乳状液稳定后，继续加入12g丙二醇甲醚，搅拌30min，制得微乳液油墨清洗剂。</p>
    <p>[0022]	实施例3 ：</p>
    <p>[0023]	在室温下，先将2g脂肪醇聚氧乙烯醚硫酸铵、2g脂肪醇聚氧乙烯醚（加成数为 10)、2g辛基酚聚氧乙烯醚、Ig十二烷基苯磺酸、2g烷基醇酰胺、2g单乙醇胺和2g脂肪醇聚 氧乙烯醚加入63g纯水中搅拌，溶解，配制成溶液，然后在搅拌条件下，加入8g乙酸丙酯进 行乳化，搅拌20min，待乳状液稳定后，继续加入18g丙二醇丁醚，搅拌30min，制得微乳液油</p>
    <p>墨清洗剂。</p>
