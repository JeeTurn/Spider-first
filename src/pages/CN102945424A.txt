      <div class="claim-text">1.	一种将防伪信息通过加密运算和信道编码生成二进制调制信号，并通过循环查表调制方式将防伪信息嵌入在整个页面中的二维加密防伪印刷方法，其特征是：对防伪信息进行数字化，生成八位一组的二进制信息表，防伪信息是图像信息、文字信息或商标信息，二进制信息表中的第i组八位二进制信息记作Ni，i为大于等于O的正整数，八位二进制加密信息记作C，C为O &lt; C &lt; 256的正整数，算符&amp;采用+、_、X、+四种运算，二位二进制控制变量记作k，控制变量k为O彡k彡3的正整数，k = O时Ni&amp;C定义为Ni+C，k = I时Ni&amp;C定义为Ni-C，k = 2时Ni&amp;C定义为NiX C，k = 3时Ni&amp;C定义为Ni+ C，设定加密参数C的初值，设定控制变量k的初值为k = O，设定二进制防伪信息表中8位二进制信息Ni的位置控制变量i = 1，从二进制防伪信息表中第一个8位二进制信息N1开始，对二进制防伪信息表中的每一个8位二进制信息进行Ni&amp;C加密运算，并且在对每一个8位二进制信息进行Ni&amp;C加密运算的同时进行k+Ι运算，使下一运算指向Ni&amp;C其中k已增加了 I，通过对二进制防伪信息表中的每一个8位二进制信息进行Ni&amp;C加密运算，生成二进制加密防伪信息表，调幅网点的形状设置为两种：□和&#9674;，其中□定义为数字O、&#9674;定义为数字1，利用生成的二进制加密信息经过信道编码，生成具有检错和纠错功能的八位一组二进制调制信号，将原始连续调图像信号经过栅格化处理（RIP)和混合加网输出半色调混合加网图像信号，其中包括调幅网点和调频网点图像信号，利用生成的二进制调制信号采用循环查表法调制方式调制混合加网图像信号中调幅网点的形状，使调幅网点的形状按照□和&#9674;有规律的发生改变，使混合加网图像信号中相邻八个调幅网点通过形状的改变携带八位二进制防伪信息，从而生成在整个页面网点中嵌入防伪信息的混合加网图像信号，实现防伪印刷。</div>

    <p>二维加密防伪印刷方法</p>
    <p>所属技术领域：</p>
    <p>[0001]	本发明涉及一种防伪印刷技术，特别是一种二维加密防伪印刷技术，该防伪印刷技术可以用于各种印刷制品的防伪。</p>
    <p>背景技术：</p>
    <p>[0002]	现有的较为普通的防伪方法有以下几种：第一种是激光防伪标志，用激光隐性油墨萤光油墨印刷技术将产品的标徽或特殊的识别图案印制成产品的防伪标贴，且同一类产品使用同一种标贴，因防伪标贴较容易伪造，而伪造的防伪标贴又被用在假冒产品上，造成产品的真假混淆，因此难于有效防伪。第二种是密码防伪标贴，其所采用的方法是每件产品编一组数码，每件产品的编码都不相同，将此数码印制在标贴上并遮盖起来，同时将此数码 存入可供消费者查询的计算机数据库中，消费者购买产品时，通过电话或入网计算机将标识上的数码输入计算机数据库进行比较识别，相同即为真，不同即为假，方法简单，识别容易，不易伪造，但实际使用中，因编码数据是计算机统一生成后印制标贴的。代表产品真假编码数据可能被非法拷贝造假，同时，编码也可回收未查询的产品上的编码造标而贴在假产品上，防伪效果难于保证。第三种是纹理防伪，以其标贴上的纹理特征防伪，虽然较难伪造，但由于只设标贴的序号码，且是明码，每枚标贴可反复查询，造假者可通过仓库保管员或售货员将标贴上的序号及查询时所反映的必要纹理特征即方格中的有无现象抄袭后按此特征批量伪造。综上所述，现有的防伪方法都存在一定的缺点，因而不能从根本防止假冒</p>
    <p>女口</p>
    <p>广叩ο</p>
    <p>发明内容：</p>
    <p>[0003]	为了克服现有的各种印刷制品防伪印刷技术存在的缺点，本发明针对现有印刷制品防伪印刷技术存在的不足对现有技术进行了改进，提出了一种二进制加密信号调制印刷品调幅网点的形状的加密防伪印刷技术，该防伪印刷技术通过调幅网点的形状的改变将防伪信息嵌入在整个页面中，可在印刷品识别时从任意一个碎片里识别防伪信息，因此具有很强抗碎性，可从根本上杜绝采用照相、扫描等非法复制行为。</p>
    <p>[0004]	本发明解决其技术问题所采用的技术方案是：对柔性版印刷混合加网中的调幅网点和调频网点分开进行处理，对防伪信息进行数字化，生成八位一组的二进制信息表，防伪信息是图像信息、文字信息或商标信息，二进制信息表中的第i组八位二进制信息记作Ni，i为大于等于O的正整数，八位二进制加密信息记作C，C为O &lt; C &lt; 256的正整数，算符&amp;采用+、_、X、+四种运算，二位二进制控制变量记作k，控制变量k为O SkS 3的正整数，k = O时Ni&amp;C定义为Ni+C，k = I时Ni&amp;C定义为Ni_C，k = 2时Ni&amp;C定义为Ni X C，k = 3时Ni&amp;C定义为Ni+ C，设定加密参数C的初值，设定控制变量k的初值为k = 0，设定二进制防伪信息表中8位二进制信息Ni的位置控制变量i = 1，从二进制防伪信息表中第一个8位二进制信息N1开始，对二进制防伪信息表中的每一个8位二进制信息进行Ni&amp;C加密运算，并且在对每一个8位二进制信息进行Ni&amp;C加密运算的同时进行k+Ι运算，使下一运算指向Ni&amp;C其中k已增加了 1，通过对二进制防伪信息表中的每一个8位二进制信息进行Ni&amp;C加密运算，生成二进制加密防伪信息表，调幅网点的形状设置为两种：□和&#9674;，其中□定义为数字O、&#9674;定义为数字1，利用生成的二进制加密信息通过循环查表法调制调幅网点，使其有规律的按照上述两种调幅网点的形状改变混合加网中调幅网点的形状，使混合加网中调幅网点的形状有规则的发生改变，调制后相邻八个调幅网点构成一组八位二进制信息，使其携带防伪信息，并使该防伪信息嵌入在整个页面网点中，能更有效地对抗基于照相机、扫描仪、电子文档等非法复制行为。通过在印刷品中非显见地嵌入可提取的防伪信息，能够为真品提供有效证明，同时具有较强的抗伪造能力，且不增加额外的防伪成本。</p>
    <p>[0005]	为解决上述的技术问题，首先对防伪信息进行数字化，生成八位一组的二进制信息表，防伪信息可以是图像信息、文字信息、商标信息等，生成八位一组的二进制信息表，对二进制信息表中的每一个八位二进制信息进行Ni&amp;C运算，生成二进制加密信息表，利用生成的二进制加密信息经过信道编码，生成具有检错和纠错功能的八位一组的二进制调制信号。信道编码可以采用循环编码、卷积编码或Turbo编码等多种形式，将原始连续调图像信号经过栅格化处理（RIP)和混合加网输出半色调混合加网图像信号，其中包括调幅网点和调频网点图像信号，利用生成的二进制调制信号采用循环查表法调制方式调制混合加网图 像信号中调幅网点的形状，使调幅网点的形状按照□和&#9674;有规律的发生改变，使混合加网图像信号中相邻八个调幅网点通过形状的改变携带八位二进制防伪信息，从而生成在整个页面网点中嵌入防伪信息的混合加网图像信号，实现防伪印刷。</p>
    <p>[0006]	在提取防伪信息时，首先采集网点图像信号，经过对调幅网点的形状的模糊识别，提取网点的边沿信号，分辨调幅网点的形状。解调调幅网点的形状信息，输出二进制调制信号。对解调输出的二进制调制信号分别进行信道解码和解密，恢复防伪信号并输出防伪信肩、O</p>
