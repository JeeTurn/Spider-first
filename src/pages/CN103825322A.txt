    <div class="abstract">本发明公开了一种能量转移式无损均衡充电电路及方法。该电路包括逻辑判断与执行模块、继电器网络、串联蓄电池组、DC／DC双向恒流模块和直流母线，继电器网络与串联蓄电池组连接，逻辑判断与执行模块与继电器网络连接，继电器网络还通过DC／DC双向恒流模块与直流母线连接；逻辑判断与执行模块实时的获取串联蓄电池组中任一节蓄电池的电压，判断该电压值是否处于异常，根据该判断结果控制继电器网络和DC／DC双向恒流模块进行无损均衡充电，均衡过程中的过渡能量暂存于直流母线中。本发明在既能从直流母线取能量恒流输出给某一节蓄电池，也能从该蓄电池恒流取能量回到直流母线，整个电路拓扑采用模块化的设计思想，提高了均衡效果，增加了系统的鲁棒性。</div>

    <p>一种能量转移式无损均衡充电电路及方法</p>
    <p>技术领域</p>
    <p>[0001]	本发明涉及蓄电池组充电均衡技术领域，更具体的说，特别涉及一种能量转移式无损均衡充电电路及方法。</p>
    <p>背景技术</p>
    <p>[0002]	近年来，储能技术发展较快，可以通过快速的有功/无功控制，有效响应负荷波动，保障重点负荷在供电系统故障情况下的应急供电。目前较广泛应用的是铅酸电池与UPS组成的电源系统作为应急电源，但由于铅酸电池的体积能量比和重量能量比太小，无法做至IJ“大容量可移动”，也没有采用智能电池管理系统，使得电池寿命缩短。随着锂离子电池的性能不断提升，其能量密度每年以10% -15%的速度递增。目前锂离子电池的质量比能量已达到100-160Wh / kg，体积比能量已达到240-350Wh / L，比镉镍、镍氢电池都高，具有高容量、高电压和无污染等特点，因而一经问世就受到人们的广泛关注。</p>
    <p>[0003]	但是一般电池的单体端电压普遍较低，整个电池组的有效容量会受最小容量单体电池的影响。若单体间的电压差异得不到及时平衡和抑制，这种不均衡性就会随电池的充放电循环不断加剧，最终会影响整个电池组的工作。</p>
    <p>[0004]	在现有技术中，常见的均衡以被动均衡和主动均衡方式为主。其中，被动均衡也称为能量耗散式均衡，均衡效率低下且损耗能量。传统的主动均衡大多使用电感或者电容为能量转移的载体，通过控制开关的切换来实现能量传递。此种主动均衡方式虽具有结构简单，易操作等优点，但存在均衡电流不可控，均衡效果差，可靠性不高等缺点。且现有均衡方式不能实现模块化设计，不能实现电池组单元间的均衡，这样就使得参与均衡的单体电池个数有限。因此，设计与制作有效的电池组均衡系统具有很强的现实意义。</p>
    <p>发明内容</p>
    <p>[0005]	本发明的目的在于针对现有的串联蓄电池组充电时没有均衡或均衡效果不佳的技术问题而提供一种能量转移式无损均衡充电电路及方法。</p>
    <p>[0006]	为了解决以上提出的问题，本发明采用的技术方案为:</p>
    <p>[0007]	&#8212;种能量转移式无损均衡充电电路，包括逻辑判断与执行模块、继电器网络、串联蓄电池组、DC / DC双向恒流模块和直流母线，所述继电器网络与串联蓄电池组连接，所述逻辑判断与执行模块与继电器网络连接，所述继电器网络还通过DC / DC双向恒流模块与直流母线连接；</p>
    <p>[0008]	所述逻辑判断与执行模块实时的获取串联蓄电池组中任一节蓄电池的电压，判断该电压值是否处于异常，并根据该判断结果控制继电器网络和DC / DC双向恒流模块进行无损均衡充电，并且均衡过程中的过渡能量暂存于直流母线中。</p>
    <p>[0009]	根据本发明的一优选实施例:所述逻辑判断与执行模块包括相互连接的主控模块和采集模块，所述采集模块分别与继电器网络和DC / DC双向恒流模块的控制输入端连接。</p>
    <p>[0010]	根据本发明的一优选实施例:所述串联蓄电池组包括正负极依次串联的多个蓄电池；所述继电器网络包括依次设于串联蓄电池组的每节蓄电池的正负极上的控制开关，以及第一和第二单刀双掷开关，所述控制开关的控制端，第一和第二单刀双掷开关的控制端均与采集模块连接，并且奇数个控制开关还与第一单刀双掷开关的上端子连接，偶数个控制开关还与第二单刀双掷开关的上端子连接，所述第一单刀双掷开关的下端子与第二单刀双掷开关的上端子连接，所述第二单刀双掷开关的下端子与第一单刀双掷开关的上端子连接，所述第一单刀双掷开关和第二单刀双掷开关的输出端均与DC / DC双向恒流模块连接。</p>
    <p>[0011]	根据本发明的一优选实施例:所述主控模块和采集模块均采用ARM架构的STM32系列微处理器，所述主控模块为STM32F107微处理器，所述采集模块为STM32F103微处理器。</p>

    <p>[0017]	根据本发明的一优选实施例:所述步骤S3中的无损均衡充电采用在同一时间只均衡串联蓄电池组中的一节蓄电池的分时均衡的方式。</p>
    <p>[0018]	根据本发明的一优选实施例:所述无损均衡充电包括抽取高电压态蓄电池的电流以降低其电压值来降低该蓄电池的SOC值，或对低电压的蓄电池Celli灌入电流使其电压升高。</p>

    <p>具体实施方式</p>
    <p>[0031]	下面结合实施例及附图对本发明作进一步详细的描述，但本发明的实施方式不限于此。</p>
    <p>[0032]	参阅图1所示，本发明提供一种能量转移式无损均衡充电电路，包括逻辑判断与执行模块301、继电器网络302、串联蓄电池组303、DC / DC双向恒流模块304和直流母线305，继电器网络302与串联蓄电池组303连接，逻辑判断与执行模块301与继电器网络302连接，继电器网络302还通过DC / DC双向恒流模块304与直流母线305连接；逻辑判断与执行模块301实时的获取串联蓄电池组303中任一节蓄电池的电压，判断该电压值是否处于异常，并根据该判断结果控制继电器网络302和DC / DC双向恒流模块304进行无损均衡充电，并且均衡过程中的过渡能量暂存于直流母线305中。</p>
    <p>[0033]	继电器网络302包括依次设于串联蓄电池组303的每节蓄电池Celli的正负极上的控制开关Kj，以及第一和第二单刀双掷开关，控制开关Kj的控制端，第一和第二单刀双掷开关的控制端均与采集模块32连接，并且奇数个控制开关Kj还与第一单刀双掷开关的上端子连接，偶数个控制开关Kj还与第二单刀双掷开关的上端子连接，所述第一单刀双掷开关的下端子与第二单刀双掷开关的上端子连接，所述第二单刀双掷开关的下端子与第一单刀双掷开关的上端子连接，所述第一单刀双掷开关和第二单刀双掷开关的输出端均与DC / DC双向恒流模块304连接，其中，i和i均为正整数。</p>
    <p>[0034]	在本实施例中，选用串联蓄电池组303包括正负极依次串联的多个蓄电池Celli，即celll-celie，它们采用正负极相连的串联方式，这时控制开关的数量即个七个，即Kl至K7，而第一和第二单刀双掷开关分别采用K8和K9表示。</p>
    <p>[0035]	在本发明中，逻辑判断与执行模块301包括相互连接的主控模块31和采集模块32，所述采集模块32分别与继电器网络302和DC / DC双向恒流模块304的控制输入端连接。</p>
    <p>[0036]	再具体的，采集模块32的一排控制端与串联蓄电池组303上的控制开关K1-K7及单刀双掷开关K8、k9相连，其下方两控制端与DC / DC双向恒流模块304相连，单刀双掷开关K8的下端子与单刀双掷开关K9的上端子相连，K9的下端子与K8的上端子相连，celll的正极连有控制开关K1，负极连有控制开关K2，以此类推，蓄电池cell6的正极连有控制开关K6，负极连有控制开关K7。奇数类的控制开关，如K1、K3、K5、K7等的另一段与单刀双掷开关Κ8的上端子相连，偶数类的控制开关，如Κ2、Κ4、Κ6等的另一段与单刀双掷开关Κ9的上端子相连，连接在串联蓄电池组303的控制开关如Κ1-Κ7，单刀双掷开关K8、k9，它们的控制端与采集模块32相连。而连接在串联蓄电池组303上的控制开关，如K1-K7，以及单刀双掷开关K8、K9均采用继电器实现。主控模块31只根据采集模块32采集到的实时串联蓄电池组303电压信号V做出逻辑判断，而采集模块32既负责上传实时串联蓄电池组303信息，又负责执行继电器网络302的通断。</p>
    <p>[0037]	所述的主控模块31和采集模块32均采用ARM架构的STM32系列微处理器，所述主控模块31为STM32F107微处理器，所述采集模块32为STM32F103微处理器，由于STM32为基于ARM构架的高性价比MCU，其处理速度可达上百兆赫兹，可以让系统满足更高的实时性要求。</p>
    <p>[0038]	本发明中额DC / DC双向恒流模块304应根据实际电路的具体应用而选择，根据具体应用可以选择市场上的通用模块，也可因需要而自行设计，其功率选择取决于系统的直流母线最高电压，本实施例中的串联蓄电池组303额定电压为384V，最高电压可达到450V，针对此实例选用的均衡电源为功率200W、450V转4V的DC / DC双向恒流模块。</p>
    <p>[0039]	本发明的充电电路采用的是分时均衡策略，通过选通继电器网络302相应的开关，实现对某一节蓄电池降压或升压操作，例如要对蓄电池celll实现减压操作，先合上开关Kl与K2，将单刀双掷开关K8、K9打到上端子，断开开关Κ3-Κ7，通过采集模块32将DC /DC双向恒流模块304置于向直流母线灌电流模式，这样蓄电池celll就会以一恒定的电流如50A被放电到直流母线中；若要对蓄电池celll实现升压操作，在控制开关设置不变的情况下，通过采集模块32将DC / DC双向恒流模块304置于向蓄电池灌电流模式，这样DC /DC双向恒流模块304从直流母线取能量向蓄电池celll以一恒定的电流如50A灌入。对蓄电池cell2的操作也和蓄电池celll的一样，只是要将单刀双掷开关K8、K9打到下端子。其它蓄电池的均衡操作方法与此类似。奇数类的蓄电池选通设置跟蓄电池celll &#8212;样，偶数类的蓄电池选通设置跟蓄电池cell2 &#8212;样。</p>
    <p>[0040]由于逻辑判断与执行模块301包括主控模块31和采集模块32两个模块。采集模块32负责采集电池组的实时信息，包括电压V、电流1、温度T等。并将这些实时信息第一时间上传给主控模块31，主控模块根据采集模块32上传的串联蓄电池组303实时信息，通过判断逻辑，下发操作指令给采集模块32，并由采集模块32的控制端通过继电器网络302选通串联蓄电池组303中某节蓄电池，同时下发指令给DC / DC双向恒流模块304，确定它的恒流方向，可快速地对选择的蓄电池实现升压或降压均衡，直流母线305在这里充当能量过渡暂存的角色，实现系统所述的无损均衡思想。其中的主控模块31负责处理信息，采集模块32负责传递信息与执行相应操作，这样的流水线处理有利于实现模块化扩容，即可以让主控模块31下挂多个采集模块32，提高系统的鲁棒性。</p>
    <p>[0041]	结合图2所示，本发明提供的一种能量转移式无损均衡充电方法，包括以下步骤，</p>
    <p>[0042]	第一步.逻辑判断与执行模块301实时的获取串联蓄电池组303中任一节蓄电池的电压，也就是采用逻辑判断与执行模块301的采集模块32实时的获取串联蓄电池组303中任一节蓄电池的电压；</p>
    <p>[0043]	第二步.逻辑判断与执行模块301判断该电压值是否处于异常，也就是采用逻辑判断与执行模块301的主控模块31判断采集模块32所采集到的电压值是否处于异常；</p>
    <p>[0044]	第三步.逻辑判断与执行模块301根据判断的结果控制继电器网络302和DC /DC双向恒流模块304进行无损均衡充电，并且均衡过程中的过渡能量暂存于直流母线305中，也就是逻辑判断与执行模块301的采集模块32根据主控模块31的判断结果控制继电器网络302和DC / DC双向恒流模块304进行无损均衡充电。</p>
    <p>[0045]	其中的无损均衡充电采用是以上所介绍的在同一时间只均衡串联蓄电池组303中的一节蓄电池的分时均衡的方式。而无损均衡充电包括抽取高电压态蓄电池的电流以降低其电压值来降低该蓄电池Celli的SOC值，或对低电压的蓄电池Celli灌入电流使其电</p>
    <p>压升闻。</p>
    <p>[0046]	具体的，在分时均衡方式完成充电后，继电器网络302断开充电回路，并且若逻辑判断与执行模块301判断单节蓄电池Celli间的最高电压与最低电压的压差超出预设值，则控制控制继电器网络302和DC / DC双向恒流模块304继续进行无损均衡充电。</p>
    <p>[0047]	参阅图3所示，为本发明所提供的充电电路的具体工作流程图:</p>
    <p>[0048]	第一步:系统上电后，首先开启串联蓄电池组303上的充电机，给串联蓄电池组303充电，此时蓄电池处于总体充电状态。</p>
    <p>[0049]	第二步:由采集模块32采集η节蓄电池的电压Vl_Vn，并立即将其上传给主控模块31，主控模块将接收的η个电压信号存储在Flash中。</p>
    <p>[0050]	第三步:主控模块31调用电压比较函数将采集到的电压Vi (i = 1，…，η)与参考电压V I比较。若已有至少一个电压Vi大于V I，说明电池组中至少有一节电池的电压上升过快，且其超过了正常范围，应对其进行放电降压处理。</p>
    <p>[0051]	第四步:此时充电机正常工作，保证其余电池正常充电，控制模块31下发指令给采集模块32，采集模块通过控制端让电池celli两端的控制开关Kj与K (j+Ι)合上，这时若j是奇数，将单刀双掷开关K8、K9合到上端子；若j是偶数，将单刀双掷开关K8、K9合到下端子，其余开关一律断开。</p>
    <p>[0052]	第五步:采集控制模块32下发指令给DC / DC双向恒流模块304，使其置于抽取电池电流到直流母线方式。这样蓄电池celli就将恒流放电给直流母线，这时采集模块32实时采集蓄电池celli的电压Vi，并将其上传给主控模块31。主控模块调用电压比较函数比较V I与Vi的大小，如果Vi大于V I，说明蓄电池celli的电压仍过高，则对其继续均衡放电；如果Vi小于V I，说明蓄电池celli已处于正常状态，则应关断均衡回路，并回到第二步重复以上流程。</p>
    <p>[0053]	第六步:在上述第三步中，主控模块通过调用电压比较函数比较V I与Vi(i =I，…，η)的大小后,若蓄 电池电压Vi (i=l,…,η)都小于参考电压V I，说明蓄电池组处于正常充电状态。[0054]	第七步:主控模块31调用电压比较函数将采集到的电压Vi (i = 1，…，η)与参考电压V II比较。若蓄电池组电压Vi (i = 1，…，η)存在至少一个不大于VII，说明蓄电池组处于正常升压状态，且未接近充满状态，并继续回到第二步，重复以上流程。</p>
    <p>[0055]	若通过电压比较函数比较电压Vi (i = 1，…，η)与V II后，发现蓄电池组所有电池电压Vi(i = l，…，η)都大于参考电压V II，说明该组蓄电池已接近充电完成状态，即该组蓄电池组上的所有电池的SOC都已接近I。</p>
    <p>[0056]	第八步:关断充电机，停止整体的充电，这一步将根据压差状态进一步的均衡电池，使电池组达到最佳的平衡状态，将之前所采集并存储的实时电压Vl-Vn进行大小排序，找到电压最大值Vk和电压最小值Vp，主控模块31调用电压比较函数将电压最大值Vk和电压最小值Vp的差值与参考压差Vth比较。</p>
    <p>[0057]	若电压最大值Vk和电压最小值Vp的差值比参考压差Vth小，可认为整体电池组中各电池电压差已在均衡误差允许范围内，则充电完成。</p>
    <p>[0058]	若电压最大值Vk和电压最小值Vp的差值比参考压差Vth大，说明电池组中各电池间的压差过大，应通过均衡控制由内部自调整来减小这一差值。</p>
    <p>[0059]	第九步:采集模块32通过控制继电器网络302，选通蓄电池cellp的均衡回路，其中，奇数类的蓄电池控制开关和单刀双掷开关的状态与celll &#8212;致；偶数类的蓄电池控制开关和单刀双掷开关的状态与cell2 &#8212;致。</p>
    <p>[0060]	第十步:采集控制模块32下发指令给DC / DC双向恒流模块304，让其置于从直流母线305取能量恒流输入到蓄电池的方式，而此时电池组中的其它电池处于放电状态，它们的能量转入直流母线并通过DC / DC双向恒流模块304回馈给电池cellp。</p>
    <p>[0061]	第十一步:主控模块31开启内部的定时器，等待定时器计时结束，然后关断均衡回路，并继续回到第二步，重 复以上流程。</p>
    <p>[0062]	上述实施例为本发明较佳的实施方式，但本发明的实施方式并不受上述实施例的限制，其他的任何未背离本发明的精神实质与原理下所作的改变、修饰、替代、组合、简化，均应为等效的置换方式，都包含在本发明的保护范围之内。</p>
