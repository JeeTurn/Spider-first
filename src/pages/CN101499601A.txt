    <p>将换向器中碳板与铜基面焊接的方法</p>
    <p>技术领域</p>
    <p>本发明属于换向器的加工方法，尤其涉及的是换向器中碳 板与铜基面的焊接方法。 背景技术</p>
    <p>目前国内外直流电机的换向器，大都采用无氧铜或银铜合 金制造，这种换向器使用寿命较短。采用碳（石墨）来制造直 流电机的换向器可以延长使用寿命，遇到最大的难题是如何将 碳片与铜可靠连接问题。最早是由德国科学家开始研究，企图 用于军事，二次大战德国战败后，德国科学家被美军俘虏，进 入美国继续进行研究。已制成碳换向器，但由于成本太高不能 推广应用，后来各国有很多专利，都用机械镶嵌式结构解决碳 与铜的连接问题，也制成了碳换向器，由于制造工艺复杂成本 较高，不能广泛应用。后由于无刷电机问世，无刷电机以长寿 命&#36111;得关注。对碳换向器的研究工作，被耽搁了 一段时间。汽 车采用电喷装置后，燃油泵由于体积限制，不能够采用无刷电 机，再由于直流电机以其特有的生产成本低、体积小、功率大， 又被得到重视。再则，由于世界各国政府为控制汽车排放污染， 同时为节省石油使用其它燃料，执行新的能源政策，在汽油中 规定添加乙醇等醇类物质。乙醇中所含的弱酸，对金属换向器</p>
    <p>造成腐蚀，使普通换向器的使用寿命在2 0 0 0_3 0 0 0小时左右，不 能满足汽车燃油泵的使用要求，为此开发新型的碳换向器又成 为世界各国关心的课题。碳换向器以其化学稳定及其自润滑的</p>
    <p>的优越特性，使得汽车燃油泵得以长寿命。碳换向器制造的关键 技术在于碳片和铜壳的可靠焊接。由于碳是非金属材料， 一 般 钎料很难润湿碳片表面，必需用碳片表面金属化或活性钎料的 方法，才能实现与铜两种材料的钎焊，表面金属化的方法用一</p>
    <p>3般钎料钎焊，制造工艺复杂，接头强度低较少&#37318;用。近十几年 来 一 些新材料对焊接技术提出了更高的要求，特别是各种新型 陶瓷、金刚石、碳复合材料等都要求进行可靠连接，为适应这 一要求钎焊技术近年来也得到飞快发展，活性钎料就是适应这 一要求发展起来的，这些钎料也可以铜和碳片可靠连接。这给 碳换向器低成本制造提供了可能。活性钎料就是在一般钎料中</p>
    <p>含有Ti 、 Zr 、 Al 、 Cr 、 P等活化物质，它可以实现陶瓷、金刚 石、碳复合材料等自身或与其它材料的可靠连接。常用的活性 钎料为Ag-Cu-Ti钎料，这种钎料由于含有较多的银，再加上含 钛钎料冶炼困难价格非常贵，很难大量应用。同时由于碳和铜 两种材料物化性能差别较大，在钎悍接头容易产生较大的内应 力。有时会造成接头破坏，这些都是本研究要解决地问题。本 专利使用Cr、 P作为活化元素，用高温加热循环减少和消除铜 壳与碳片间的内应力，较好地解决了低成本制造碳换向器的难 题。碳换向器使用领域相当广阔，除可以涵盖所有的直流电机 外，还可以在航空、航天、军工等领域推广应用。目前，美国、 德国、日本、墨西哥等国已成功生产了碳换向器，碳换向器已 在国际上，被广泛采用在汽车燃油泵生产中，但这项技术 一 直 被国外保密、垄断。根本见不到公开报道。 发明内容</p>
    <p>鉴于背景技术存在的问题，本发明要解决的技术问题是提 供 一 种低成本的钎料，使得换向器焊接后其使用寿命长碳板与 铜基面的焊接方法。</p>
    <p>为此，本发明是采用以下技术方案来实现的： 一 种将换向 器中碳板与铜基面焊接的方法，其特征在于：</p>
    <p>a 、钎料成分：采用粉状或箔状的活性钎料作为焊接钎料， 其成分重量比例为   Cu:   5 &#8212; 20%  ,   P:   5 &#8212; 15%  ,   Sn:   3_10%， Cr: 5_2 0%，   Ti和Si微量，其余为Ni;</p>
    <p>b 、焊前准备：将碳板基面清理干净，同时将铜基面清洗干净，并晾干备用，用工装夹具由下至上按铜基面一焊接钎料</p>
    <p>&#8212;碳材料的顺序组装定位好，然后放置于真空炉内； c 、焊接：采用钎焊工艺进行真空钎焊，工艺参数为：真空 度不低与   5xl0 &#8212; 3pa，由室温升温到   3 0 0 °C&#8212;5 0 0°C，升温 时间3 0 &#8212;5 0分钟，保温1 0 &#8212;2 Q分钟，继续升温到6 5 (TC &#8212;8 5 0 °C ,升温时间3 0 &#8212; 5 0分钟，再保温1 0 &#8212;2 0分钟，再继续升 温到9 0 0 °C &#8212; 1 0 0 0 °C ，升温时间1 0_2 0分钟，又保温15 &#8212; 30 分钟，然后稳步降至常温，钎焊时施加 一 定的压力，范围 在0.     <p>实施例1:这种将换向器中碳板与铜基面焊接的方法，先对 钎料进行配比制作，钎料成分采用粉状的活性钎料作为焊接钎 料，每份钎料按100克计，其中Cu含10克，P含10克，Sn含 5克，Cr含15克，Ti和Si分别为0. 0 0 3克，其余为Ni;称取 上述成分进行混合，即成粉状钎料，然后作焊前准备，也即是 将碳板基面采用丙酮清洗清理干净，同时将铜基面清洗干净， 并晾干备用，用工装夹具由下至上按铜基面 一 焊接钎料 一 碳材 料的顺序组装定位好，然后放置于真空炉内；接下来就是焊接 工艺，焊接时&#37318;用钎坪工艺进行真空钎焊，工艺参数为：真空 度为4 x 1 0 &#8212;3Pa ，由室温升温到4 0 (TC ,升温时间4 5分钟，保温 1 5分钟，继续升温到7 0 0 °C ，升温时间4 0分钟，再保温1 5分 钟，再继续升温到9 5 (TC ,升温时间1 5分钟，又保温2 0分钟， 然后稳步降至常温，z降焊时施加 一 定的压力，压力为范围在0.05 Mpa 。</p>
    <p>5按照上述工艺对碳板和铜基面进行焊接后，制成的换向器</p>
    <p>经过测试，其焊接强度为5 Mpa ,连接电阻为2 . 1亳欧，绝缘电 阻为11 0. 6M Q , 耐高温和^氐温性能均超过9小时，其成本下降 了 2 1倍，使用寿命达到1 0 5 0 0小时。完全满足碳换向器的制作 要求及性能指标。</p>
    <p>实施例2 :本实施例同样是先对钎料进行配比制作，钎料成分采 用粉状的活性钎料作为焊接钎料，每份钎料也按100克计，其 中Cu含5克，P含10克，Sn含7克，Cr含13克，Ti和Si分 别为0. 0 02克，其余为Ni;称取上述成分进行混合，即成粉状钎 料，将粉状钎料进行压注，就可成为箔状钎料，本实施例采用 的是箔状钎料，然后作焊前准备，也即是将碳板基面采用丙酮 清洗清理干净，同时将铜基面清洗干净，并晾干备用，用工装 夹具由下至上按铜基面 一 焊接钎料 一 碳材料的顺序组装定位 好，然后放置于真空炉内；接下来就是焊接工艺，焊接时采用 钎焊工艺进行真空钎焊，工艺参数为：真空度为5xl0 &#8212; 3pa,由 室温升温到5 0 (TC ,升温时间5 0分钟，保温1 8分钟，继续升温 到8 0 0 °C ，升温时间4 5分钟，再保温1 8分钟，再继续升温到 1 000 °C ，升温时间1 8分钟，又保温2 5分钟，然后稳步降至常 温，钎焊时施加一定的压力，压力为范围在0. 08 Mpa。</p>
    <p>按照上述工艺对碳板和铜基面进行焊接后，制成的换向器 经过测试，其焊接强度为6. 2 Mpa,连接电阻为1. 6毫欧，绝缘 电阻为11 5 . 4M Q ,耐高温和低温性能均超过1 0小时，其成本下 降了 2 3倍，使用寿命达到1 0 1 0 0小时。也完全满足碳换向器的 制作要求及性能指标。</p>
    <p>综上所述，可见，本发明的方法对于焊接碳板和铜基面来 说，具有很好的效果，并极大地降低了成本，提高了换向器的 使用寿命，必将给制作换向器行业带来积极的效果。</p>
