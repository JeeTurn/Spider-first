    <p>[0002]	Web容器是服务器端容器，主要是为了使原来各孤立的站点之间的信息能够相互通信、共享而提出的一种接口，其所使用的是因特网antemet)上统一、开放的标准， 如 HTTP (HyperText Transfer Protocol，超文本传输协议）、XML(Extensible Markup Language，可扩展标记语言）、SOAP (Simple Object Access Protocol，简单对象访问协 τΧ ) &gt; WSDL (Web Services DescriptionLanguage, Web Service 描述i吾 g )等，所以Web容器可以在任何支持这些标准的环境（例如：Wind0WS，Linux)中使用。</p>
    <p>[0003]	Web容器主要功能是提供网上信息浏览服务，负责管理和运Rkrvlets (&#8212;种服务器端的Java应用程序)、JSP (JavaServer Pages 一种实现普通静态HTML和动态HTML混合编码的技术）页面、过滤器和事件监听器等组件，这些组件是遵循Java EE(SUN公司推出的企业级应用程序版本）规范的标准Web组件。目前Web应用于J2EE(Java 2platform enterprise edition, Java 2企业版本平台）平台，该平台提供了一个基于组件的方法来设计、开发、装配以及部署企业应用程序，该平台还提供多层分布式应用模型、组件复用和客户端远程的分布式对象的框架。</p>
