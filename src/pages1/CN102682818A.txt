    <p>重放装置、重放方法及程序</p>
    <p>技术领域</p>
    <p>[0001 ] 本发明涉及一种重放装置、重放方法及程序。</p>
    <p>背景技术</p>
    <p>[0002]	下述重放装置为世人所知：用户从多个内容中选择、决定任意的内容，重放内容信息。但是存在以下情况：因存在多个内容，所以用户从该多个内容中找出所需的内容需要时间。因此，例如专利文献I公开了以下方法：从多个内容中，部分重放用户选择的内容信息，通过用户听取该部分重放的内容信息，有效地找出用户所需的内容。</p>
    <p>[0003]	专利文献I :特开2006-344342号公报 发明内容</p>
    <p>[0004]	但是，在专利文献I公开的构成中，部分重放用户选择的内容信息的位置（地点）是确定的，根据内容不同，存在用户无法判断部分重放的内容是什么内容的情况。因此，要求新的可有效地找出用户所需的内容的方法。</p>
    <p>[0005]	本发明鉴于以上情况而出现，其目的在于提供一种通过容易地变更使内容信息部分重放的位置（地点），可有效地找出用户所需的内容的重放装置、重放方法及程序。</p>
    <p>[0006]	为实现上述目的，本发明的第I观点涉及的重放装置的特征在于具有：</p>
    <p>[0007]	存储部，存储多个内容各自的信息；</p>
    <p>[0008]	显示控制部，进行控制，使表示上述内容的内容识别图像显示到显示部；</p>
    <p>[0009]	指示控制部，指示上述内容识别图像的任意一个；</p>
    <p>[0010]	位置决定部，当指示了上述显示的内容识别图像的任意一个的位置时，根据对上述内容识别图像的显示范围的指示位置，决定内容的重放位置；</p>
    <p>[0011]	对象决定部，决定作为重放对象的内容；</p>
    <p>[0012]	重放控制部，从上述决定的重放位置开始对上述决定的内容进行重放。</p>
    <p>[0013]	上述重放控制部可对上述决定的内容从上述决定的重放位置开始重复重放规定时间。</p>
    <p>[0014]	上述指示控制部可在对决定的内容从上述决定的重放位置开始重放过程中出现其他内容的指示输入时，指示上述其他内容。</p>
    <p>[0015]	本发明的第2观点涉及的重放方法的特征在于：</p>
    <p>[0016]	存储多个内容各自的信息，</p>
    <p>[0017]	进行控制，使表示上述内容的内容识别图像显示到显示部，</p>
    <p>[0018]	指示上述内容识别图像的任意一个，</p>

    <p>具体实施方式</p>
    <p>[0039]	以下说明本发明的实施方式。以下记载的实施方式用于进行说明，并不限制本发明的范围。因此，只要是本领域技术人员，可采用将该各要素或全部要素置换为与其均等的要素的实施方式，这些实施方式也包含在本发明的范围内。</p>
    <p>[0040]	本发明的实施方式I涉及的重放装置是通过重放多媒体信息（例如声音信息、影像信息、文字信息、及它们的复合信息），将该信息提供给用户的装置。重放装置100如图I所不，由控制部110、存储部120、显不部130、输入部140、输出部150构成。</p>
    <p>[0041]	控制部110例如由CPU (Central Processing Unit :中央处理单元）等计算处理电路、作为工作区的RAM (Random Access Memory :随机存储器）、各种驱动器等构成,控制重放装置100的各部分，并且根据存储部120中收容的规定的动作程序，执行下述各处理。控制部110如该图所示，由指示控制部111、位置决定部112、对象决定部113、重放控制部114、显示控制部115构成。并且，重放装置100内的各构成分别连接到控制部110，各构成间的数据收发等借助控制部110进行。</p>
    <p>[0042]	指示控制部111在决定应重放的内容前，从多个内容的任意一个开始控制内容的指示。其中，内容例如是指由乐曲、影像构成的信息。指示控制部111根据来自用户的输入指示（输入部140)，从多个内容开始控制一个内容的指示。[0043]	位置决定部112决定对通过指示控制部111指示的内容部分重放的重放位置。稍后论述决定内容的重放位置的方法。</p>
    <p>[0044]	对象决定部113从多个内容的任意一个开始决定重放位置，决定将用于进行部分重放的内容作为重放对象。其中，决定是指，将通过指示控制部111指示的内容，作为进行部分重放的内容而选择的行为。并且，部分重放是指，对由规定时间（例如5分钟）构成的内容的一部分（例如I分钟）进行重放的行为。通过对象决定部113决定了作为重放对象的内容时，该内容被部分重放。</p>
    <p>[0045]	重放控制部114进行控制，使通过对象决定部113作为重放对象决定的内容重放。被控制来重放的内容通过显示部130、输出部150输出（重放）。</p>
    <p>[0046]	显示控制部115进行控制，在显示部130上显示规定的图像。</p>
    <p>[0047]	存储部120例如由ROM (Read Only Memory :只读存储器）、闪存等规定的存储装置 构成，除了收容控制部110执行的动作程序（应用程序）外，还收容各处理的执行所需的数据、通过各处理的执行取得或生成的数据等。重放装置100的处理通过控制部110执行存储部120中存储的动作程序来实现。</p>
    <p>[0048]	并且，存储部120存储多个内容以规定顺序（例如五十音顺序、记录顺序、用户确定的顺序）确定的播放列表、乐曲、影像构成的多个多媒体信息。各多媒体信息中包括乐曲/影像的标题彳目息、声首彳目息、影像彳目息、记录时间（录首/录像时间）彳目息等。</p>

    <p>[0072]	通过以上处理，可从多个内容中指示任意的内容，容易地变更部分重放该内容的重放开始时间，通过用户听取该内容的部分重放，可有效地找出用户所需的内容。</p>

    <p>[0081]	接着，决定了作为重放对象的内容、并决定了重放时间时（步骤Slll 是”)，控制部110控制位置决定部112及重放控制部114，仅以在步骤S201中决定的重放时间部分重放内容（步骤SI 12)。图9中，控制部110对与内容的内容识别图像202c (内容C)对应的内容，从0:00开始到0:30为止部分重放。</p>
    <p>[0082]	通过以上处理，可从多个内容中指示任意的内容，容易地变更部分重放该内容的重放时间，通过用户听取该内容的重放，可有效地找出用户所需的内容。</p>
    <p>[0083]	此外，本发明不限于上述实施方式，可进行各种变形及应用。</p>
